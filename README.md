Project Description

This project is a Tic Tac Toe game developed using Java Swing and MySQL database.

The application includes:

* Login system
* Tic Tac Toe gameplay
* Player statistics
* Top 5 scorer leaderboard

Features

* Login using database
* Play Tic Tac Toe against computer
* Record wins, losses, draws, and score
* Display player statistics
* Display Top 5 scorers

Database

Database: MySQL

Table:

* players

Fields:

* id
* username
* password
* wins
* losses
* draws
* score

* Main.java

* LoginFrame.java
* MainMenuFrame.java

* GameFrame.java
* StatisticsFrame.java
* TopScorersFrame.java

How to Run

1. Start MySQL and Apache.
2. Import database.
3. Open project in IntelliJ IDEA.
4. Run Main.java.
